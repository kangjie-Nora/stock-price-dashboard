# 🚀 快速安装指南（无需 Homebrew）

## 最简单的方法：直接从官网下载（推荐 ⭐⭐⭐）

**这是最简单的方式，不需要安装任何额外工具！**

### 步骤：

1. **访问 Node.js 官网**
   - 打开浏览器，访问：https://nodejs.org/
   - 您会看到一个绿色的"LTS"按钮（推荐版本）

2. **下载安装包**
   - 点击绿色按钮下载（macOS 会自动选择 `.pkg` 文件）
   - 文件名类似：`node-v20.x.x.pkg`

3. **安装**
   - 双击下载的 `.pkg` 文件
   - 按照安装向导操作（一直点击"继续"即可）
   - 可能需要输入 Mac 密码

4. **验证安装**
   - **重要**：关闭当前终端窗口，打开**新的终端窗口**
   - 运行以下命令：
     ```bash
     node --version
     npm --version
     ```
   - 如果显示版本号（如 `v20.11.0`），说明安装成功！✅

5. **安装项目依赖**
   ```bash
   cd "/Users/lukangjie/Desktop/Stock Price Dashboard"
   npm install
   ```

6. **运行项目**
   ```bash
   npm run dev
   ```

完成！就这么简单！🎉

## 备选方案：使用 nvm（类似虚拟环境，可选）

如果您想要类似 Python 虚拟环境的功能，可以使用 nvm（不需要 Homebrew）：

```bash
# 1. 安装 nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 2. 重新加载配置（或关闭终端重新打开）
source ~/.zshrc

# 3. 安装 Node.js
nvm install --lts
nvm use --lts

# 4. 验证
node --version
npm --version
```

## ❓ 常见问题

**Q: 安装后还是显示 `command not found`？**
A: 关闭终端窗口，重新打开一个新的终端窗口即可。

**Q: 下载很慢怎么办？**
A: 可以访问国内镜像：https://npmmirror.com/mirrors/node/

**Q: 安装需要多久？**
A: 通常 1-2 分钟就完成了。

**Q: 需要卸载旧版本吗？**
A: 不需要，新版本会自动覆盖旧版本。
